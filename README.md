## Project setup

``` node
// 根目录下
npm install

// server 目录下
npm install
```

### Run frontend server

``` node
npm run frontend
```

### Init PostgreSQL & Run backend server

向项目组成员获取测试数据库登录信息。

启动后端服务器：

``` node
npm run backend
```

之后您可以：

- 用户组：使用 `2020091202000` 作为账号， `123123` 作为密码登录。
- 处理者组：使用 `2018091202000` 作为账号， `123123` 作为密码登录。

### Compiles and minifies for production

``` node
npm run build
```

### Lints and fixes files

``` node
npm run lint
```
